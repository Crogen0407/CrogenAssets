CrogenHierarchy(v1.0.4)

// Get Start
1. set the initial hierarchy-specific settings for the component via "Window > Crogen > CrogenHierachy > GlobalSetting".
1. “Window > Crogen > CrogenHierachy > GlobalSetting”를 통해 컴포넌트의 초기 계층별 초기 셋팅값을 설정합니다.

2. Add a "HierarchyInfo" component to set the visuals of a specific object's Hierarchy!
2. “HierarchyInfo” 컴포넌트를 추가하여 특정 오브젝트의 Hierarchy에서 보이는 비주얼을 셋팅하세요!

3. Customize your Hierarchy window with various options like Background, Icon, TextColor, and more!
3. Background, Icon, TextColor 등의 다양한 옵션으로 자신만의 Hierarchy창을 뽐내세요!
